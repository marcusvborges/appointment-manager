# Medical Appointment Manager

Sistema fullstack para agendamento de consultas médicas, criado como monorepo visando agilidade, facilidade de teste e documentação centralizada. O projeto foi idealizado seguindo conceitos de clean architecture e boas práticas de desenvolvimento.

Projeto desenvolvido como desafio técnico.

## Tech Stack

### Backend

- Node.js 22 LTS
- NestJS
- pnpm
- TypeORM
- PostgreSQL
- JWT + Passport
- Swagger
- Docker & Docker Compose
- Zod

### Frontend
- Vue.js + Quasar

## Estrutura do Projeto

/backend → API NestJS  
/frontend → Aplicação Vue + Quasar
